This directory contains several script files to provide a quick walkthrough of the SolidFire Tools for PowerShell module.  
You should be able to complete the entire series in less than an hour.

Requirements
Section 1 - SolidFire system to make calls against with access to add/remove objects like volumes and accounts.
Section 2 - SolidFire system to make calls against with access to query information.
Section 3 - SolidFire system to make calls against with access to add/remove objects like volumes and accounts.
            vSphere environment with a minimum of one ESXi host and vCenter server with administrator privileges on the ESXi host.
